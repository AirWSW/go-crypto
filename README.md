# go-crypto

### Usage

- `github.com/AirWSW/go-crypto/aes`: Advanced Encryption Standard (aesCipher)
- `github.com/AirWSW/go-crypto/des`: Data Encryption Standard (desCipher, tripleDESCipher)

### Unit tests

```bash
$ go test ./...

# ok      github.com/AirWSW/go-crypto/des 1.414s
```
